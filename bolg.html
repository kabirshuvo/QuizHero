Q:1 Can you tell me how .map(), .filter(), .forEach(), and .find() are different in JavaScript?

  • .map(): creates a new array by applying a function to each element of the original array and it Returns something.
  • .filter(): creates a new array with all elements that pass a certain condition, based on a function that returns a boolean value.
  • .forEach() iterates over all elements of an array and applies a function to each element.
  • .find() returns the first element in an array that satisfies a certain condition, based on a function that returns a boolean value.

In summary, .map() and .filter() both creates a new array, whereas .forEach() and .find() do not.
.forEach() applies a function to each element without modifying the array, while .map() and .filter() both modify the array or create a new one. Finally, .filter() returns an array of elements that pass a certain condition, while .find() returns only the first element that passes the condition.
